# Risky Bubble

労働法上のリスクがある台詞を、「セーフ」または「危険」で判定する学習ゲームです。  
問題、解説、画像、音声、回答時間などを編集できる専用ビルダーを同梱しています。

## 主な機能

### ゲーム

- 授業モード：回答後に正解と解説を確認
- テストモード：制限時間内に「セーフ」「危険」を選択
- 時間切れ時は自動的に「セーフ」を選択
- 3ステージ制
- 正解数、誤判定、見逃し、コンボを記録
- 終了画面で間違えた問題を復習

### ビルダー

- 問題・正解・カテゴリ・解説の追加、編集、削除
- CSVの読み込みと書き出し
- 背景、左右キャラクター、報酬イラストの変更
- キャラクターの大きさ、横位置、縦位置の変更
- BGM、正解SE、間違いSEの変更と試聴
- 全問題共通の回答時間を1～30秒で設定
- ゲーム画面と報酬画面のプレビュー
- 編集結果をゲーム本体と素材フォルダへ一括保存

## 起動方法

### ゲームを遊ぶ

`index.html` をChromeまたはEdgeで開いてください。

### ビルダーを使う

1. `builder.html` をChromeまたはEdgeで開きます。
2. 「ゲームフォルダを開く」を押します。
3. `index.html` と `builder.html` が入っているフォルダを選択します。
4. 問題またはスタイルを編集します。
5. 「ゲームへ保存」を押します。

ビルダーではブラウザのFile System Access APIを使用するため、最新版のChromeまたはEdgeを推奨します。

## フォルダ構成

```text
Risky-Bubble/
├─ index.html
├─ builder.html
├─ README.md
├─ キャライラスト/
├─ 背景イラスト/
├─ 報酬イラスト/
├─ 音声/
└─ 問題CSV/
   └─ questions.csv
```

## 問題CSV

`問題CSV/questions.csv` で問題を管理します。

| 列 | 内容 |
|---|---|
| `id` | 通し番号 |
| `question` | 質問・台詞 |
| `answer` | `risk`（危険）または `safe`（セーフ） |
| `category` | 労働法上のカテゴリ |
| `explanation` | 回答後に表示する解説 |

CSVを直接編集する場合は、UTF-8で保存してください。

## GitHub Pagesでゲームを公開する

1. このプロジェクトの内容をGitHubリポジトリへアップロードします。
2. GitHubのリポジトリで `Settings` → `Pages` を開きます。
3. 公開元として対象ブランチのルートを選択します。
4. 発行されたURLへアクセスするとゲームが表示されます。

ビルダーはローカルのゲームフォルダを書き換える用途を想定しています。公開サイト上のファイルを直接更新するものではありません。
